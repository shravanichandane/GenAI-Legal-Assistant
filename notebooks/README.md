# Notebooks Directory
This directory holds Jupyter notebooks for exploration and experimentation.

## Naming Convention
Use numbered prefixes for ordering:
```
01_data_exploration.ipynb
02_clause_classification_baseline.ipynb
03_legal_bert_fine_tuning.ipynb
04_rag_evaluation.ipynb
```

> **Rule:** Notebooks are for *exploration*, not production code.
> Once a notebook produces useful logic, refactor it into `app/`.
